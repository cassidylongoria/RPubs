# R Data Analysis & Visualization Projects
This repository contains selected R Markdown projects focused on data analysis, visualization, and reproducible reporting. Each project includes the original `.Rmd` source file, required datasets, and a rendered HTML report.

## Summarizing Data: Character Strings
Demonstrates data summarization and manipulation techniques in R utilizing RStudio's internal datasets, with a focus on handling and analyzing character string variables. 

- Published R Markdown: https://rpubs.com/Komosko/1348873
- Report: `docs/summarizing-data-character-strings.html`
- Source: `notebooks/summarizing-data-character-strings.Rmd`

## Utilizing AI to Visualize Data
Analyzes electric vehicle (EV) registration data across the United States to identify trends and distribution patterns. The project demonstrates data importation from CSV files, aggregation, and visualization, while highlighting how AI tools were used to assist with data preparation and workflow efficiency. 

- Published R Markdown: https://rpubs.com/Komosko/1382763
- Report: `docs/utilizing-ai-to-visualize-data.html`
- Source: `notebooks/utilizing-ai-to-visualize-data.Rmd`
- Data: 
    - `data/EV_Cleaned.csv`
    - `data/Merged_EV_Population_2023.csv`
- External Sources:
    - EV Registration Data: https://afdc.energy.gov/data/10962
    - U.S. State Population Data (U.S. Census Bureau via Statista): 
        https://www.statista.com/statistics/183497/population-in-the-federal-states-of-the-us/
- Certificate: `docs/assets/CertificateOfCompletion.pdf`
